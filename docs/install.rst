Install
=======

Using ``pip``::

    pip install drf-api-checker

Go to https://github.com/saxix/drf-api-checker if you need to download a package or clone the repo.


|pkg| does not need to be added into ``INSTALLED_APPS``


.. _test_suite:


How to run the tests
--------------------

.. code-block:: bash

        $ pip install tox
        $ tox
